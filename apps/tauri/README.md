# Arco OS — Tauri shell

Cross-platform prototype shell for Arco. Loads the **shared root Vite build** (`../../dist`) or the dev server at `:4610`.

## Dev

From repo root (recommended):

```bash
npm run tauri:dev:all
```

Or run server + web yourself, then:

```bash
npm run tauri:dev
```

## Build

```bash
npm run tauri:build
```
