Install Arco OS

Android app for phones and Chromebooks with Android app support.

Chromebook: use the ZIP download below if APK is blocked. Rename the file to arco-os-mobile.apk after download.
Download ZIP (Chromebook — recommended) Download APK (Android phone)
Setup
  1. Open this page with HTTPS:
  2. If you see “Your connection is not private” → Advanced → Proceed.
  3. Download ZIP (Chromebook) or APK (phone).
  4. Files → Downloads → rename .zip to .apk if needed → tap to install.
  5. Keep your Mac running npm run dev:chromebook on the same Wi‑Fi.
