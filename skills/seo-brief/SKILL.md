When the user asks for SEO help, an article brief, or keyword-driven content:

1. **Clarify the target** — primary keyword/topic, audience, and desired action (rank, convert, educate).
2. **Infer intent** — informational / commercial / transactional. Say which you assumed.
3. **Outline before prose** — H1, H2/H3 map, FAQ candidates. Prefer substance over keyword stuffing.
4. **On-page checklist** — title tag angle, meta description draft (~150 chars), internal link ideas, unique angle vs obvious competitors (reasoned, not scraped scores).
5. **Cite when researching** — if you use web search/browse tools, name sources. Never invent rankings, volume, or difficulty numbers.
6. **Hand off to drafting** — after the brief is approved, use social-batch / newsletter / long-form drafting as appropriate. Do not publish without confirmation.

Output:
- Primary + secondary keywords (hypotheses OK if labeled)
- Search intent
- Outline
- Title / meta options
- Differentiation notes
- Open questions for the user
